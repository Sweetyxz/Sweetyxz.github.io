    【基础知识】HTML-DOM节点

    DOM结点

    • 整个文档是一个文档节点

    • 每个 HTML 元素是元素节点

    • HTML 元素内的文本是文本节点

    • 每个 HTML 属性是属性节点

    • 注释是注释节点

      HTML DOM 方法

      我们可以在节点（HTML 元素）上执行的动作。

    HTML DOM 属性

    我们可以在节点（HTML 元素）设置和修改的值。

    nodeValue - 节点的值

    • 元素节点的nodeValue 是 undefined 或 null
    • 文本节点的 nodeValue 是文本本身
    • 属性节点的 nodeValue 是属性值

    nodeType - 节点的类型

    返回节点类型

    元素类型 NodeType
    元素 1
    属性 2
    文本 3
    注释 8
    文档 9
